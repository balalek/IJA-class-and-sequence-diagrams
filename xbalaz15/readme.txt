Cílem projektu je načítat, modifikovat a ukládat 1 diagram tříd a k němu více sekvenčních diagramů v GUI.

Členové týmu:

protože vytvořený výsledný jar soubor není spustitelný.

Verze:	Java SDK 11
	JavaFX 17
